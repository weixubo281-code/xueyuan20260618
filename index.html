Curriculum

6个阶段

6节课分别对应6个阶段，每节完成一个明确成果。

第1课｜AI视觉产品设计

从文字到商业级视觉图

学会商业视觉提示词公式，掌握主体、场景、光影、质感、构图和风格统一。

  • ✓ 产品主视觉生成
  • ✓ 局部重绘 / 换背景 / 扩图
  • ✓ 导入Figma做成封面或海报
课堂产出：一张小红书 / 朋友圈可发布的商业产品海报。
第2课｜AI UI/UX界面设计

从需求到一套APP页面

用AI梳理功能结构、页面清单、用户流程，再落到Figma核心界面。

  • ✓ APP信息架构
  • ✓ 首页 / 功能页视觉方向
  • ✓ 卡片、按钮、图标、留白细节
课堂产出：一套APP核心页面，至少3页。
第3课｜AI 3D产品视觉

从AI图到3D产品感渲染

不讲复杂建模，重点跑通AI概念图、AI建模、Blender渲染的最小流程。

  • ✓ 产品概念图生成
  • ✓ 简单3D模型流程
  • ✓ 材质、灯光、相机基础设置
课堂产出：一张AI+3D产品展示图。
第4课｜AI动态视觉与短视频

把静态设计变成15秒视频

学习静态图转视频的镜头语言，让作品更像商业广告，而不是AI乱动。

  • ✓ 推拉摇移与景深
  • ✓ 15秒产品动画分镜
  • ✓ 海报动态化 / Logo动态化
课堂产出：一个15秒产品 / Logo / 海报动态视频。
第5课｜设计师AI工作流

把AI变成你的设计助理

解决日常工作提效：拆需求、写方案、做竞品、写设计说明、整理作品集。

  • ✓ 方案与设计说明生成
  • ✓ 竞品分析与页面逻辑
  • ✓ 个人提示词库搭建
课堂产出：一个个人AI设计工作流模板。
第6课｜AI个人IP与商业变现

把作品变成账号内容和接单能力

把课堂作品包装成可发布内容，建立个人IP定位、内容选题和服务报价。

  • ✓ 小红书 / 抖音内容选题
  • ✓ 标题、正文、标签和封面
  • ✓ AI设计服务报价包装
课堂产出：个人IP定位卡 + 3条选题 + 基础接单报价表。

FAQ

常见问题

没有AI基础可以学吗？

可以。课程会从提示词结构、工具使用逻辑和案例演示开始，但前提是你愿意跟着做练习与作业。

每节课多长时间？

建议每节90—120分钟，包含方法讲解、老师实操、学员练习、点评答疑和作业布置。

学完能不能直接接单？

课程会帮你搭好作品基础与服务包装思路，但真正稳定接单还需要持续打磨案例、发布内容和优化报价。第6课会重点讲个人IP与接单方向。

后续想学更深怎么办？

如果你后续还想继续深入，可以继续进入更高阶的作品集深化、商业项目实战或针对性辅导内容，逐步扩展自己的能力边界。
